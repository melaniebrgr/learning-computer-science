# WebGPU

> At a certain level, WebGPU is a very simple system. All it does is run 3 types of functions on the GPU: Vertex Shaders, Fragment Shaders, and Compute Shaders. With that you can draw triangles, points and lines to textures, and run computaions on the GPU. (3)

> If you just want to get something on the screen you’re far better off choosing a library that provides the large amount of code you’re going to have to write when doing it yourself. (3)
