# InfoVis

## LECTURE 5

Data might be raw, but the treatment never is. It is squewed in some way towards the story we want to tell. Data can be broken down into univariate, bivariate, and multivariate. 

Univariate can be represented in more than one dimension but only exists in one.

Bivariate data can be plotted on an x and y axis.

Multivariate data can be plotted on three axises. A map is an example: terrain, politics separations, elevation, roads, etc. another example is a heat map marking fixations.

You can continue to build upon something to add variables to it.

Types of data
- Categorical (noun)
- Nominal (quantifies)

Tufte says the we should always supply context, e.g. Compared to what? Causality? The world is multivariate.
Tufte principles
- Micro/macro reading
- Layering and seperation. 1+1= 3. Use colour and other Gestalt principles to stratify. Things that are not important should be receded or eliminated.
- Small multiples. Small objects where everything is consistent except the data. Possible because the eye can take in a lot of data in a small space
- Narrative of time a space
