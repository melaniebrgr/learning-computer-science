### InfoVis tool kits
Tool kits include illustrator, photoshop, excel, SPSS.

### The big fish software tool kits
FREE
- Processing and processing.js
- Google visualization API: google chart wizard, google chart tool, birdeye (although may not be that thriving), public data site (based on gap minder)
- D3, a JavaScript library for visualizing data that succeeds Protovis. See Mike Bostock's website.
- Many Eye, don't need to code but they own it
- Circos developed by Martin Krzyzinski (?)

COMMERCIAL
- Thinkmap. Useful for node networks.
- Tableau. Has free trial version. Tableau public
- Amira. Geared towards volume rendering.

### The little fish
- Prefuse - a java based toolkit.
- PrefuseFlare - for visualizations in Flash
- Polymaps - JavaScript library for maps
